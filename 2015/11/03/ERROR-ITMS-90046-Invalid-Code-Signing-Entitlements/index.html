使用Xcode 7 打包上传app时候环境没变突然报错，如下：

ERROR ITMS-90046: "Invalid Code Signing Entitlements. Your application bundle's signature contains code signing entitlements that are not supported on iOS. Specifically, value 'development' for key 'aps-environment' in 'Payload/app' is not supported."

解决方案：clean 工程然后重新打包上传。

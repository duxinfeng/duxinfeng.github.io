Mac上 GitHub Pages + Hexo 搭建 Blog

发表于2015-05-01|更新于2021-04-27|Blog
|字数总计:226|阅读时长:1分钟|阅读量:

环境安装

  1. 安装Node.js:官方https://nodejs.org/下载pkg安装包
  2. 执行 npm install hexo-cli -g
  3. 新建blog文件进，终端cd blog进入到这个文件夹,hexo init
  4. hexo s如果没报错，基本服务安装完成

配置_config.yml文件

_config.yml有两个，一个博客根目录下_config.yml基本文件，一个主题的_config.yml文件

常用命令

  1. hexo new ‘Hello Word’ 创建一个一片文章模板，到/source/_post 目录下找到Hello Word.md的文件进行编辑 (hexo n ‘Hello Word’)
  2. hexo generate #生成静态页面至public目录 (hexo g)
  3. hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server） (hexo s)
  4. hexo deploy #将.deploy目录部署到GitHub(hexo d)

An error occurred while processing the post-install hook of the Podfile.

2016年4月11日

执行Pod install 报错：

An error occurred while processing the post-install hook of the Podfile.

undefined method `project’ for #

解决方案：

打开Podfile,把里面的 project 改为 pods_project

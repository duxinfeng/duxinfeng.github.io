碎碎念在2016年3月21日

3月 21, 2016 分類 生活

又来写了，距离上一次碎碎念已经50天了，中间开篇过两次，不知道写啥，然后放弃了。只有当你静下心来，你才能很好的思考。

1、经实验表明，21天并不能养成一个习惯，你需要一直坚持，坚持让他成为生活中不可缺少的一部分。一件事如果你一直坚持，那么他或许成为一种习惯，倘若中间持续中断几天，那么你这习惯就被破坏了。去年的时候有段时间强迫自己看书，即使有时事情，也会强迫自己去看几分钟或者看几页，大概坚持了四十多天的样子，后来断了几天，到现在就再也没看了，那段时间买了七八本书，五六本还在睡觉。

2、思考。我认识的一些优秀的人里面他们都有一个共同特点就是善于思考，能很清楚表达自己观点，讲的内容有理有据，能从一个你想不到的角度来解释一个现象或一件很平常的事情。即使在出牛逼也能让你一脸懵逼的觉得很有道理。

3、思考需要给自己自己营造气氛。12年十二月份搬到公司附近，省去每天地铁到公司的70min车程，现在每天从家走到工位15min足够。原本是每天能节省两个小时可以去干更多自己的事情，然而并没有，甚至少了阅读和思考。在地铁上你会发现搭建都在看自己的手机或者kindle，所以你也在看你的手机，有时候会看一些公共账号，有时候会用微信阅读看看书，有时候看着看着就在思考一件事情，想明白一个道理，或是记下来。然而，现在坐车的时间省下来了，阅读少了，思考少了。

4、依然经营不好爱情。和别人的区别就是别人的女朋友和别人的男朋友。彼此都存在问题，都在和以前比，没用以前XXX。

5、凌晨苹果春季发布会，然而并不关心，也不想熬夜去看了，身体素质越来越差，快要死了。明早看个新闻就好。

6、API调用者。上周看到的一句话，做了三年的开发，然后发现只是API调用者，和比人比拼谁知道的API多，谁用的熟。

7、当一件事情只有发生在你身边的时候，才感觉到紧急。今年房价疯涨一下，然后也意外知道几个同学房买好了，心里会惊讶一下了，才开始去细想这个事情。之前有个错误的认识就是放假将来会跌，好幼稚。房价会随着人才的流动而变化，而且每年都在印钞票，物价上涨，怎么可能会跌，她会是一种向上涨的一个过程，不可能会像股市那样跌宕起伏。当然不排除风水不好地势不佳的地方房价会降，然而也不回去那。

8、钱解决99%的事情，所以多挣钱。

9、不要懒，早起，吃早饭。早睡！

10、没有10.

DOUAudioStreamer设置currentTime无效

1月 31, 2016 分類 iOS

关于DOUAudioStreamer

DOUAudioStreamer是基于豆瓣团队Core Audio实现的用于iOS/Mac的开源流媒体播放引擎。代码托管地址：https://github.com/douban/DOUAudioStreamer

示例Demo

1、官方有个demo:https://github.com/douban/DOUAudioStreamer/tree/master/example/DOUASDemo
2、另一个功能比较完善的demo:https://github.com/Aufree/ESTMusicPlayer

遇到的问题

因为需求问题，我需要在初始化DOUAudioStreamer对象时，就去设置currentTime属性：

_streamer = [DOUAudioStreamer streamerWithAudioFile:track];
_streamer.currentTime = 10.0; 
[_streamer play];

然而，这并不像我们相信那样，从10s的时候开始播放。

猥琐的解决办法

当然如果你正常的播放音频文件，然后拖动滑动条从某个时间点开始播放是没有问题的，完全正常。也就是说streamer在play之前有个加载音频文件的缓冲时间。所以想到一个比较猥琐的办法，

_streamer = [DOUAudioStreamer streamerWithAudioFile:track];
[_streamer play];
[NSThread sleepForTimeInterval:0.5];
_audioStreamer.currentTime = 10.0;

嗯，貌似草草的解决了。

如果看过demo的知道，demo里会有几个KVO的方法，其中有一个是监听 updateSliderValue方法，监听_audioStreamer当前播放进度。

所以又意淫出另外一个方法，如果self.audioStreamer.currentTime < 10, 我就设置self.audioStreamer.currentTime = 10;,

初始化

然后，在updateSliderValue方法里做判断

if (self.audioStreamer.currentTime<10)
{
     self.audioStreamer.currentTime = 10;
}

后记

当然遇到这样问题，也去Github上提交了issue(https://github.com/douban/DOUAudioStreamer/issues/102)，但是一直没有回复，也没有精力去研究源码，改一改，也没用那水平，所以就这样解决了。

也想过找另外类似的播放引擎，确实找打一个StreamingKit：https://github.com/tumtumtum/StreamingKit
确实也简单易用，但是和DOUAudioStreamer存在同样的问题，最终还是选择了DOUAudioStreamer。

NSString *usrstring = @"xxxxxx";
NSURL *URL = [NSURL URLWithString:usrstring];

有时候你会发现URL为nil;为什么呢？
因为你的usrstring里包含中文或者其他什么文字，需要进行一下UTF8转码

NSURL *URL = [NSURL URLWithString:[urlstring stringByAddingPercentEscapesUsingEncoding:NSUTF8StringEncoding]];

集成支付宝出现的一堆warning:AlipaySDK4Standard/AlipaySDK/Library/UTDID.framework

1月 04, 2016 分類 iOS

警告信息

warning: (i386) /Users/scmbuild/workspace/standard-pay/IOS/cp_record_client_release_515040/1212/mspc_iphone_msdk/AlipaySDK4Standard/AlipaySDK/Library/UTDID.framework/UTDID(UTDIDBaseUtils.o) unable to open object file: No such file or directory

解决办法

1) Go to Build Settings -> Build Options -> Debug Information Format

2) Change the Debug setting from “DWARF with dSYM File” to “DWARF”

3) Leave the Release setting at “DWARF with dSYM File”

AlipaySDK.framework 是先 build 的静态 lib ，然后转成的 framework ，但是在 build 静态 lib 时 CLANG_ENABLE_MODULES 没有被设置成 disabled 。
